    🎀 CloudConvert在线压缩

    使用在线工具可以方便的压缩字体

    网址：https://cloudconvert.com/

    如图所示，可以选择压缩为WOFF或WOFF2格式

    然鹅，我用这个压缩的NotoSerifSC字体不能用，其他字体如EBGaramond、Lato、Monoton、monaco等可以正常使用

    字体 压缩前 压缩后 压缩比
    Lato.ttf 115KB 33KB 28.70%
    Monoton.ttf 48.73KB 20KB 41.04%
    monaco.ttf 65KB 32KB 49.23%
